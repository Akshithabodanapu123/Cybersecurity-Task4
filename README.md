# Task 4 - Exploitation & System Security

## Objective
The objective of this task is to understand penetration testing methodology, exploitation techniques, password attacks, social engineering, malware analysis, and system hardening.

## Topics Covered

- Reconnaissance
- Scanning
- Exploitation
- Post Exploitation
- Password Attacks
- Social Engineering
- Malware Analysis
- System Hardening

## Tools Studied

- Kali Linux
- Nmap
- Metasploit Framework
- John the Ripper
- Metasploitable2

## Learning Outcomes

- Understood penetration testing phases.
- Learned vulnerability assessment concepts.
- Studied password attack techniques.
- Learned malware analysis fundamentals.
- Understood system hardening best practices.
