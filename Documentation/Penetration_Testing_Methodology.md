# Penetration Testing Methodology

## Reconnaissance

Reconnaissance is the process of gathering information about the target system.

Examples:

- IP Address Collection
- Network Information
- Service Discovery

## Scanning

Scanning is used to identify open ports and services running on a target system.

Tool:

- Nmap

## Exploitation

Exploitation is the process of using a vulnerability to gain access to a system.

Tool:

- Metasploit Framework

## Post Exploitation

After exploitation, information is collected to understand the impact of the vulnerability.

## Reporting

The findings are documented along with recommendations for improving security.
